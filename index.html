HSSP CURVE

The current implementation works with Blast output version BLASTP 2.2.29+. It is important to note that the values presented here come from a sample output which main contain bad data (test it with your own). All whats implemented is referenced from this paper.

Features:
    Ability to window the data within a given range
    Ability to define the distance from the hssp curve (0 being the original hssp curve)
